Todos los formularios de la web manejan metodos para el envio de las diferentes peticiones que realiza el usuario. Considerando la arquitectura Cliente-Servidor, el manejo de las peticiones se hace por medio del protocolo http y actualmente https, cada petición debe ser atendida en un tiempo de respuesta conforme al modelo TCP/IP. Los metodos principales de los formularios son:

    • Get:

      POST:

      Este metodo se encarga de enviar la información para poder registrar un cambio o un nuevo registro dentro del sistema, enviandola igual por el metodo estandar; pero mediante un ocultamiento, esto significa que no es visible para el usuario, lo que nos ofrece un mayor grado de seguridad para el manejo de información
